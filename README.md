# Project Management CLI

## Description

A Python command-line application for managing users, projects, and tasks.

## Features

- Add users
- Create projects
- Assign tasks
- Mark tasks complete
- Save data to JSON
- Load saved data
- Object-Oriented Programming
- Uses the Rich package for CLI output

## Project Structure

```
project-management-cli/
│
├── models/
├── utils/
├── data/
├── tests/
├── main.py
├── requirements.txt
└── README.md
```

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python3 main.py
```
